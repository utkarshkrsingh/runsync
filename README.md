### Note:

Runsync currently supports only four languages.
- Go
- C 
- C++
- Python

### Building process

1. Clone the repo.
```bash
git clone https://utkarshkrsingh/runsync.git $HOME/Downloads/runsync
```

2. Build it.
```bash
cd $HOME/Downloads/runsync
make build 
```

*Runsync is currently only available on Linux.*
